Lighten Up, World
A general-purpose data visualisation project that represents data on an abstract map of the world.

Sources include live data from external weather APIs and pre-collected time-lapse data. The software for the map is run from a Raspberry Pi, and can be controlled via a React.js WebApp, which connects to a lightweight WebSocket server. All written in C and all open source.

ARM11 Emulator and Assembler

1st year project at ICL, written in C. The assembler translates an ARM (subset of) assembly source file into a binary file. The emulator is capable of simulating the execution of this binary.
